# Advanced Animated Calculator

## Project Overview

Advanced Animated Calculator is a modern web-based calculator developed using HTML, CSS, and JavaScript. This calculator provides standard arithmetic operations along with scientific functions, animated background effects, calculation history, and responsive design.

## Features

* Basic arithmetic operations (+, -, ×, ÷)
* Scientific functions:

  * Square Root (√)
  * Square (x²)
  * Cube (x³)
  * Factorial (n!)
* Animated falling numbers background
* Sliding history panel
* Stores calculation history using localStorage
* Copy result to clipboard
* Theme toggle support
* Keyboard shortcuts support
* Responsive user interface
* Backspace and clear functions
* Error handling for invalid calculations

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Keyboard Shortcuts

* Enter → Calculate result
* Backspace → Delete last character
* Escape → Clear display
* Numbers / Operators → Direct input
